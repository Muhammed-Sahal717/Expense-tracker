# Expense Tracker

A simple **Expense Tracker web application** built using **HTML, CSS, and Vanilla JavaScript**.  
This project helps track expenses, calculate total spending, and show category-wise counts.

---

## Features

- Add expenses with:
  - Name
  - Amount
  - Category
- Display a list of all expenses
- Automatically calculate **total expense**
- Show **category-wise expense count**
- Dynamic UI updates without page reload

---

## Technologies Used

- **HTML5**
  - Semantic structure
  - Forms and inputs
- **CSS3**
  - Flexbox layout
  - Responsive design
  - Clean UI styling
- **JavaScript (ES6+)**
  - Arrays & Objects
  - DOM Manipulation
  - Event Handling
  - Array methods:
    - `.map()`
    - `.reduce()`
  - `Object.entries()`

---

## JavaScript Concepts Practiced

- State management using arrays
- Rendering UI from data
- `map()` for transforming data to HTML
- `reduce()` for:
  - Calculating total expenses
  - Counting categories
- Object to array conversion using `Object.entries()`
- Preventing form reload with `event.preventDefault()`
